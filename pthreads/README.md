### Compilação e execução
1. Abra o terminal

2. Compile com o comando

   `gcc -pthread nome_arquivo.c -o nome_saida`   

3. Execute com o comando

   `./nome_saida`   

Para códigos-fonte na linguagem C++, basta substituir o `gcc` por `g++`.
